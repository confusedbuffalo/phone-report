Do not edit the files in this folder.

This folder contains poly files for boundaries of the regions used in the reports.

The files are automatically updated by the `fetch-polys.js` script.

Files in this directory are sourced from OpenStreetMap data via [https://polygons.openstreetmap.fr/](https://polygons.openstreetmap.fr/) and are licensed under the [ODbl](https://opendatacommons.org/licenses/odbl/)
